### cfbfastR.nfl.load_nfl_rosters(seasons: List[int])
Load roster data

Example:

    nfl_df = cfbfastR.nfl.load_nfl_rosters(seasons=range(1999,2021))

Returns:

    pd.DataFrame: Pandas dataframe containing rosters available for the requested seasons.
